# 🎬 OTT Finder India

A web app to search movies & TV series and find where to watch them
on Indian OTT platforms like Netflix, Prime Video, JioCinema, and Hotstar.

## 🚀 Features (Building Day by Day)
- [ ] Search movies & series
- [ ] View details (poster, genre, rating)
- [ ] OTT platform availability in India
- [ ] Watchlist (localStorage)

## 🛠️ Tech Stack
- HTML, CSS, JavaScript
- TMDB API
